# JENECONK Integrated Global Solutions Ltd Website

Static, GitHub-hostable website for the JENECONK technology ecosystem.

## Pages

- `index.html` - Landing page
- `about.html` - Company vision and ecosystem structure
- `solutions.html` - Solution areas
- `training.html` - Training programs
- `contact.html` - Contact form and business identity
- `education-suite.html` - Education Suite launch placeholder
- `business-suite.html` - Business Suite placeholder
- `cbt-portal.html` - CBT Portal placeholder

## Preview

Open `index.html` in a browser, or run a local static server from this folder.

## Deployment

This site can be deployed directly to GitHub Pages, Vercel, Netlify, or any static hosting provider.
